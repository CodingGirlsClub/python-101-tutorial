下面任选一题，或者有自己的想法也可以在教练的指导下另外选题。

选题 1

假如我想把 xkcd.com 上的所有漫画图片爬下来。能写一个 Python 脚本完成吗？

提示：

  • 页面的链接有什么规律？
  • 如何用 Python 循环和字符串拼接遍历所有的链接？
  • 漫画图片的选择器是什么？
  • 如何用 Python 提取 <img> 标签的 src 属性？
  • Python 如何保存文件？
  • 保存多个文件如何命名？
  • 输出些什么让你知道程序正在运行？

进阶：

每个图片都有一个 title 属性，能否把它也顺便保存下来？
