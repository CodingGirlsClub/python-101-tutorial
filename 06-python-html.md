# HTML

HTML 是超文本标记语言（hypertext markup language）的缩写。

（常见语病：「HTML 语言」。「L」和「语言」同义重复了，直接说「HTML」即可）



# HTML 的构造

    <!DOCTYPE html>
    <header>...</header>
    <body>...</body>
    </html>

常见元素

- 块 `<div>`
- 行内 `<span>`
- 链接 `<a>`
- 图片 `<img>`
- 视频 `<video>`
